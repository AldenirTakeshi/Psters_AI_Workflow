# Claude Workspace Helpers

This folder contains project-specific helpers to maintain and release `psters-ai-workflow`.

## Included

- `commands/develop-plugin.md` — implementation workflow for plugin maintenance.
- `commands/release-plugin.md` — release workflow with version/changelog checks.
- `agents/plugin-submission-reviewer.md` — final quality gate before commit/release.
- `agents/plugin-release-manager.md` — release orchestration guide.
